◈ Technique Theory

Cursed Spirit Manipulation

The technique operates on one principle: consume a cursed spirit and it becomes a permanent asset. The spirit retains all properties. You retain full control. The arsenal expands indefinitely.

This repository follows the same logic. Every challenge absorbed is documented for redeployment — same vulnerability surface, same attack vector, same exploit pattern. Each writeup is a spirit catalogued in exact detail: how it was found, how it was dismantled, what it leaves behind in the arsenal. The compendium doesn't forget.

◈ Deployed Techniques

Arsenal — Cursed Tool Registry

In jujutsu, cursed tools are ranked by how much cursed energy is infused into them — and how impossible they are to counter. The arsenal here follows the same structure. Some tools nullify entire categories of defense. Others are workhorses you reach for every engagement. All of them are catalogued and ready to deploy.
Special Grade Cursed Tools No binding necessary — nullifies all conventional defenses
Custom Exploit Development
Inverted Spear of Heaven — nullifies all defenses, bespoke to target
Hand-written shellcode, kernel exploits, novel attack chains with no public PoC. Built for a single target. Cannot be patched against before it lands.
pwntools (advanced)
Split Soul Katana — cuts through the soul of a binary
Full ROP chain automation, heap grooming, ret2libc — used at this tier when the binary has no obvious entry, only a structural weakness to exploit precisely.
Custom C2 / Reverse Shells
Playful Cloud — a tethered weapon with total reach
Bespoke command-and-control channels, encrypted callbacks, anti-detection payloads. Deployed when standard shells get caught immediately.
Grade 1 Cursed Tools Extraordinarily powerful — only sorcerers of equal rank can wield them effectively
Ghidra / IDA Pro
Dragon-Bone Sword — carved from a structure that was never meant to be seen
Full decompilation, cross-reference graphs, FLIRT signatures, scripting APIs. Takes experience to use at full power — but shows you everything once you do.
Metasploit (custom modules)
Pot of Four Virtues — contains anything, releases anything
A container for any payload. Off-the-shelf modules are Grade 2. But custom-written Metasploit modules shaped to a specific CVE are something else entirely.
Frida / dynamic instrumentation
Slumber Bead — hooks into consciousness mid-execution
Runtime injection, function hooking, API tracing across both mobile and desktop targets. Reaches inside running processes and rewires them in real time.
Grade 2 Cursed Tools Reliably powerful — standard-issue for any serious engagement
Burp Suite
Binding Vow Lance — intercepts and rewrites the contract
Web proxy, scanner, repeater, intruder, decoder. The entire HTTP surface is yours to inspect and manipulate. No web engagement goes without it.
nmap + scripts (NSE)
Projection Sorcery Bow — maps the entire domain from range
Full network topology, service versions, OS fingerprinting, default-credential checks via NSE scripts. The first thing cast before any engagement.
sqlmap
Cursed Blade of the Origin — finds the flaw in the foundation
Automated SQL injection detection and exploitation across all major database backends. Blunt but effective — the database rarely survives contact.
Semi-Cursed & Non-Cursed Tools No innate cursed energy — but in the right hands, lethal all the same
Wireshark / tcpdump
A sorcerer's ear pressed to the wire — hears everything
Full packet capture, protocol dissection, stream reconstruction. No cursed energy required — just the ability to read what was never meant to be read.
hashcat / john
Attrition. Not a weapon — a grindstone.
Rule-based, mask, combinator, and dictionary attacks. Pure compute against a captured hash. The lock breaks eventually if the wordlist is right.
ffuf / gobuster
Domain Expansion: Infinite Directory
Virtual-host fuzzing, endpoint brute-force, parameter discovery. The attack surface is always larger than the application wants to admit.
volatility / autopsy
Reverse Cursed Technique — recovers what was already destroyed
Memory forensics, process dumps, registry hives, artifact recovery. What was deleted is still there. What was running is still visible. The past isn't gone.
